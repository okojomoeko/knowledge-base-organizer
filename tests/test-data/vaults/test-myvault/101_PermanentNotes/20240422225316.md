# サーバー名表示(Server Name Indication、SNI)

- [[20240422225316|SNI]]によりサーバ名ごとに[[SSL/TLS]][[証明書]] を設定できるようになり、リクエストから[[証明書]] を適用するサーバ名を判断して適用するようになる
    - つまり、1つのサーバーに対して1つしか使えなかった[[SSL/TLS]][[証明書]] が[[ドメイン]] 単位で利用できるようになる
    - [1つのサーバーで複数の証明書が使える、SNIって？｜SSL基礎知識｜Zenlogic - 株式会社IDCフロンティアのレンタルサーバー](https://www.idcf.jp/rentalserver/user-support/knowledge/ssl/sni.html)
